使用hexo和github pages搭建博客

搭建步骤

  • 下载安装hexo
  • 在github上创建github page仓库

    采坑记录

  • 注意_config.yml的文件格式，每个key:后面需要有个空格，否则会报错
  • 更改_config.yml文件中的站点名称，需要重新启动hexo server，否则不会生效
  • 在github创建github pages仓库，需要把仓库名命名为username.github.io，public类型，私有的目前是需要收费的
  • github pages仓库建好后，需要在setting->options->github pages，中选择theme，否则访问会一直报404
  • 敲命令hexo deploy后，报错“TypeError: source.replace is not a function”，把_config.yml中的message: []这一项删掉就好了。
  • deploy后，通过username.github.io访问不生效，需要等个1分钟才会生效，有延时
